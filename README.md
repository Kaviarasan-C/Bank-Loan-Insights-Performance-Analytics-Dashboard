🏦 Bank Loan Analysis Dashboard (Power BI)
📌 Project Overview

The Bank Loan Analysis Dashboard is an interactive Power BI project designed to analyze loan applications, funding, repayment trends, and borrower characteristics.
This dashboard helps financial institutions understand loan performance, identify risk factors, and make data-driven decisions.

The project transforms raw loan data into meaningful insights using Power BI visualizations, DAX measures, and data modeling.

📈 Key Insights

Majority of loans are successfully repaid (Good Loans ~85%).

Debt Consolidation is the most common reason for loans.

Most borrowers prefer 36-month loan terms.

Higher risk loan grades have significantly higher interest rates.

The majority of borrowers live in rented homes.
